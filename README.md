# Aisha Website - Admin Management System

## Overview
This project is a web-based application developed during my internship at Talal Abu-Ghazaleh. It focuses on building an administrative system that allows managing users, news, and content efficiently.

The goal of this project was to apply real-world web development concepts and create a functional system with a structured backend and user-friendly interface.

---

## Features
- Admin dashboard for managing system data
- User management (add, edit, delete users)
- News management system
- Role-based permissions handling
- Form validation and data handling
- Multi-page navigation using ASP.NET Web Forms

---

## Technologies Used
- ASP.NET Web Forms
- C#
- SQL Server
- HTML, CSS, JavaScript
